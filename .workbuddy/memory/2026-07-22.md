# 2026-07-22

## 开发：初中英语口语评测与打卡网站
- 技术栈：Node + Express（后端，JSON 文件存储于 data/db.json）+ 原生前端（public/）。
- 功能：创建/加入班级（6位班级码）、班级成绩榜（同班学生互见成绩）、口语评测（Web Speech API 识别+评分，含不支持浏览器时的手动输入降级）、每日打卡（10分钟最低时长，服务端强制校验）、打卡记录与一键导出 Excel（xlsx 库，明细+汇总两表）。
- 数据模型：classes / students(token 鉴权) / scores / checkins。
- 启动：`node server.js`（端口 3000），已在后台运行。
- 说明：测试中文字符在 Git Bash 终端被编码损坏，属测试环境问题，应用本身正确处理 UTF-8。

## 重大更新（v2）：角色 + 真实发音评测 + 课堂互动
- 数据库：改用 Node 内置 `node:sqlite`（DatabaseSync，文件 data/app.db），无需联网装原生模块。注意启动时会有 ExperimentalWarning，属正常。
- 腾讯云语音评测：封装 tencentcloud-sdk-nodejs 的 soe.v20180724（DescribeSyncSpeechEvaluation），由 env TENCENT_SECRET_ID/KEY/REGION 控制；未配置时自动退回本地逐词评分。前端按 /api/assess/config 切换录音方式（云端录 16k PCM WAV / 本地用 Web Speech 或手动输入）。
- 角色：老师（建班设密码，teacher_token 鉴权）与学生（班级码+姓名加入）。学生禁止任何设置类接口（已做 403 校验）。
- 新增功能：语音作业(assignment)发布与提交评分；上课点名（随机/顺序/按小组，分组过滤+校验）；单词抢答小游戏（老师加词库、抽词、点名抢答、记分排行榜）。
- Excel 导出：打卡记录（明细+汇总）与单作业成绩，均教师可导出。
- 测试：建班/加入/作业提交/成绩榜/点名分组过滤/单词抢答记分/角色403/两种Excel导出/打卡10分钟校验，全部通过。
- 注意：浏览器录音(getUserMedia/Web Speech)需 localhost 或 https 安全上下文。
